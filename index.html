Ajuda e Explicações


O que são números binários, decimais e outras bases?

Os números binários são representados por dois símbolos: 0 e 1. Eles são a base para o sistema de computação digital.

O sistema decimal é o sistema de numeração padrão que usamos no cotidiano, baseado em 10 símbolos: 0-9.

Outras bases como octal (base 8) e hexadecimal (base 16) são utilizadas em computação para representar números de forma compacta.

Como as conversões funcionam?

Para converter de binário para decimal, você usa a fórmula de soma dos valores das potências de 2. Exemplo: 101 (binário) = 1×2² + 0×2¹ + 1×2⁰ = 5 (decimal).

Operações suportadas:

Você pode realizar operações de soma, subtração, multiplicação e divisão entre números binários ou decimais, dependendo do modo selecionado.
